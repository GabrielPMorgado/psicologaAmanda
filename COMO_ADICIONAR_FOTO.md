# Como adicionar a foto da Amanda Morgado

## Passo a passo:

1. Salve a imagem que você enviou no chat
2. Renomeie a imagem para: `amanda-morgado.jpg`
3. Coloque a imagem nesta pasta:
   ```
   c:\Users\gabri\OneDrive\Área de Trabalho\psicologaAmanda\public\images\amanda-morgado.jpg
   ```

## Alternativa: Use o Explorer do Windows

1. Abra o Explorer do Windows
2. Navegue até: `c:\Users\gabri\OneDrive\Área de Trabalho\psicologaAmanda\public\images`
3. Cole a imagem nesta pasta
4. Renomeie para: `amanda-morgado.jpg`

A imagem será exibida automaticamente no site! ✨

## Formato da imagem:
- Nome: amanda-morgado.jpg
- Formato recomendado: JPG ou PNG
- Tamanho recomendado: Mínimo 500x500px (quadrada é ideal)
